# KPIs de Projeto

| KPI | Fórmula | O que mede |
|-----|---------|------------|
| SPI (Schedule Perf. Index) | EV / PV | Aderência ao prazo |
| CPI (Cost Perf. Index) | EV / AC | Aderência ao custo |
| Velocity | Σ story points concluídos por sprint | Capacidade do time |
| Lead Time | Data conclusão − data solicitação | Tempo total de entrega |
| Cycle Time | Data conclusão − data início | Tempo em execução |
| % SLA cumprido | Tickets no SLA / total | Qualidade da sustentação |
| Escopo aceito | Entregas aceitas / planejadas | Qualidade de escopo |
