# EVM — Earned Value Management

## Grandezas
- **PV** Planned Value — valor planejado até a data
- **EV** Earned Value — valor agregado (entregue)
- **AC** Actual Cost — custo real

## Índices
| Índice | Fórmula | Interpretação |
|--------|---------|---------------|
| SV (Variação de prazo)  | EV − PV | > 0 adiantado |
| CV (Variação de custo)  | EV − AC | > 0 abaixo do orçamento |
| SPI                     | EV / PV | 1 = no prazo |
| CPI                     | EV / AC | 1 = no orçamento |
| EAC (Estimate at Completion) | BAC / CPI | Custo total previsto |
